## Install

#### With yarn

```sh
yarn add @getpingback/ui
```

#### With npm

```sh
npm install @getpingback/ui
```

## Getting started

Import the component and just use it.

```jsx
import { Badge } from "@getpingback/ui";

const App = () => {
  return <Badge>Badge Component</Badge>;
};
```
